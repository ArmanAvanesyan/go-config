# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please **do not open a public issue**. Report it privately so we can address it before public disclosure.

**Contact:** [security@example.com](mailto:security@example.com)

Include the following in your report:

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

We will acknowledge your report promptly and keep you informed as we work toward a fix.
