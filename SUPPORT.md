# Support

## Documentation

Start with the [README](./README.md) and the `examples/` directory for usage guidance.

## Questions

Open a [GitHub Discussion](../../discussions) for general questions or usage help.

## Bugs

Use the [bug report template](.github/ISSUE_TEMPLATE/bug_report.yml) to file a bug. Please include reproduction steps, your Go version, and OS.

## Feature Requests

Use the [feature request template](.github/ISSUE_TEMPLATE/feature_request.yml) to propose new functionality.

## Security Issues

Do not open a public issue for security vulnerabilities. See [`SECURITY.md`](./SECURITY.md) for the responsible disclosure process.
